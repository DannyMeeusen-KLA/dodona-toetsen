Schrijf een programma waarbij de gebruiker een willekeurig aantal getallen ingeeft op één lijn, gescheiden door een spatie.  

Het programma telt het aantal even getallen in een lijst en drukt dit aantal af.  

### Voorbeeld

```console?lang=python&prompt=>>>
>>> 3 7 8 2
2
>>> 5 32 13 18 206 6
4
>>> 1 3
0
>>> -4
1
```
