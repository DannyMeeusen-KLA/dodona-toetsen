Schrijf een programma waarbij de gebruiker twee gehele getallen kan ingeven:  
- de ondergrens van een geheel interval,  
- de bovengrens van een geheel interval;.  

Het programma vormt een lijst met de kwadraten van alle getallen in dit gesloten interval.  
Het programma drukt deze lijst af.


### Voorbeeld

```console?lang=python&prompt=>>>
>>> 5
>>> 7
[25, 36, 49]
>>> 0
>>> 3
[0, 1, 4, 9]
```
