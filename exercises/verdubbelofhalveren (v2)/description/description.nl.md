Schrijf een programma waarbij de gebruiker een willekeurig aantal getallen ingeeft op één lijn, gescheiden door een spatie.  

Het programma maakt een lijst aan en voegt de getallen één-per-één toe waarbij:  
- een even getal gehalveerd wordt,  
- een oneven getal verdubbeld wordt.  

### Voorbeeld

```console?lang=python&prompt=>>>
>>> 4 6 7 58 13 12
[2, 3, 14, 29, 26, 6]
>>> -6 -9 5 48 23 -7 0 -2 3
[-3, -18, 10, 24, 46, -14, 0, -1, 6]
>>> 6
[3]
>>> 5
[10]
>>> 37 94 13 61 -61 -54
[74, 47, 26, 122, -122, -27]
```
