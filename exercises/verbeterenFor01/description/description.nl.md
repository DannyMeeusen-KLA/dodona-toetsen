Je krijgt een oplossing van onderstaande opgave, jammer genoeg met syntax- en/of logische fouten.  
Test het programma in de sandbox, corrigeer de fouten en dien je gecorrigeerde oplossing in.  

De gebruiker geeft één getal in met een decimaal gedeelte (een *float* dus).  
Het programma drukt het eerste cijfer na het decimaalteken (een punt) af. Als het ingevoerde getal geen punt bevat, dient een '-' afgedrukt te worden.    
  
De voorgestelde oplossing maakt gebruik van een *for*. Als je wil, mag je het herschrijven met een *while*.  

### Voorbeelden

```console?lang=python&prompt=>>>
>>> 2.3456
3

>>> 45
-

>>> 5.0
0

>>> -5.4321
4

```
