Schrijf een functie `isRechthoekig` die onderzoekt of een getal rechthoekig is.  
Een getal, groter dan 1, is rechthoekig als het geschreven kan worden als het product van twee opeenvolgende natuurlijke getallen.  
  
Als een getal rechthoekig is, dan geeft de functie de waarde **True** terug.  
Als een getal niet rechthoekig is, dan geeft de functie de waarde **False** terug.  
  
## Voorbeelden

```console?lang=python&prompt=>>>
>>> isRechthoekig(2)
True
>>> isRechthoekig(3)
False
>>> isRechthoekig(6)
True
>>> isRechthoekig(10)
False
>>> isRechthoekig(12)
True
>>> isRechthoekig(1)
False
>>> isRechthoekig(0)
False
>>> isRechthoekig(-1)
False
```
