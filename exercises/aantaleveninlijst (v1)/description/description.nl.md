Schrijf een programma waarbij de gebruiker:  
- eerst ingeeft hoeveel getallen hij wil ingeven (=*n*),    
- daarna *n* getallen ingeeft;  

Het programma telt het aantal even getallen dat ingegeven werd en drukt dit aantal af.  

### Voorbeeld

```console?lang=python&prompt=>>>
>>> 3
>>> 7
>>> 8
>>> 2
2
>>> 5
>>> 32
>>> 13
>>> 18
>>> 206
>>> 6
4
>>> 1
>>> 3
0
>>> 1
>>> -4
1
```
