In dit programma geeft de gebruiker een ondergrens in. Dit is een negatief getal en een veelvoud van -10 (je hoeft dit niet na te kijken).  
Vervolgens zal het programma alle negatieve veelvouden van -10 afdrukken, groter of gelijk aan de ingebrachte ondergrens.  
Het eerste getal dat dus afgedrukt zal worden, is de ondergrens, het laatste getal dat nog afgedrukt wordt, is 0.  
  
Indien een ondergrens groter dan 0 wordt ingegeven, dan verschijnt de tekst *De ondergrens is te hoog.*  

Je kan dit programma schrijven door gebruik te maken van een **for** of een **while**. Je mag zelf kiezen.  
Bekijk de voorbeelden.    

### Voorbeeld 1
```console?lang=python&prompt=>>>
>>> -10
-10
0
```

### Voorbeeld 2
```console?lang=python&prompt=>>>
>>> -30
-30
-20
-10
0
```

### Voorbeeld 3
```console?lang=python&prompt=>>>
>>> 10
De ondergrens is te hoog.
```
